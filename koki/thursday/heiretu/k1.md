## 並列化
* 依存関係

95%並列可
プロセッサ数: N
1 / ((1 - 0.95) + 0.95 / N)
if N = 10:  6.9%
if N = 100: 16.8%
if N = 1000: 19.6%
if N = 10000: 20.0%

50%並列可
プロセッサ数: N
1 / ((1 - 0.50 / N) / N + 0.50 / N)
